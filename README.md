Smart Laboratory Equipment Reservation System

Overview

The Smart Laboratory Equipment Reservation System is designed to help students and staff reserve laboratory equipment easily and efficiently. It prevents overbooking, tracks equipment usage, and improves laboratory management.

Features

- Equipment booking and reservation
- User login and authentication
- Prevents double booking
- Tracks usage time of equipment
- Admin dashboard for management
- Reports for heavily used and underutilized equipment
- Safe and organized booking system

Technologies Used

- HTML
- CSS
- JavaScript
- Database (if used)

How to Run

1. Download the project files
2. Open the project folder
3. Run the main file (index.html / server file)

Future Improvements

- Email notifications
